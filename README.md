📊 MIS Reporting & SLA Performance Dashboard (Excel Project)
📌 Project Overview

This project is an interactive MIS Reporting & SLA Performance Dashboard built in Microsoft Excel to track operational efficiency and SLA compliance.

The dashboard provides real-time insights into performance metrics such as SLA adherence, resolution time, and team productivity.

🎯 Objectives

Monitor SLA compliance (Met vs Breached)

Track ticket resolution and response time

Analyze monthly performance trends

Evaluate team-wise productivity

Provide management-level KPI reporting

🛠 Tools & Techniques Used

Microsoft Excel

Pivot Tables & Pivot Charts

Power Query (Data Cleaning & Transformation)

Advanced Excel Formulas:

IF

COUNTIFS

SUMIFS

VLOOKUP / XLOOKUP

Conditional Formatting

Data Validation (Dropdown filters)

Interactive Dashboard Design

📈 Dashboard Features

SLA Performance Summary

Monthly Trend Analysis

Team-wise Performance Comparison

KPI Cards (Total Tickets, SLA %, Breach %, Avg Resolution Time)

Dynamic Filters (Date / Team / Priority)

📊 Key Insights Generated

Identified peak breach months

Highlighted high-performing teams

Reduced manual reporting effort

Enabled faster decision-making through visual summaries
